# 👑 King's Crown Barbershop — Landing Page

A premium single-page landing site for **King's Crown Barbershop** located in Madras, Oregon. Built as a fully self-contained HTML file with embedded assets — no frameworks, no dependencies, no build tools required.

---

## 🗂 Project Structure

```
kings-crown-barbershop/
├── kings-crown.html        # Main landing page (fully self-contained)
├── README.md               # This file
```

> All images are base64-encoded directly into the HTML. No external image files are required to run the site.

---

## ✨ Features

- **Single-file deployment** — one HTML file contains everything: styles, scripts, and all images encoded inline
- **Cinematic hero section** — AI-enhanced shop photo with Ken Burns slow zoom and warm ring light flicker animation
- **Scroll reveal animations** — sections fade and drift into view as the user scrolls
- **Fully responsive** — optimized for desktop and mobile with `dvh` viewport fix for mobile browsers
- **Live booking integration** — all booking CTAs link directly to the Vagaro booking page
- **Click-to-call** — phone number links trigger a native call on mobile
- **Google Maps integration** — directions link in both the nav and About section
- **Instagram section** — dedicated block linking to `@kingscrownbarbershop`
- **Sticky navigation** — frosted nav bar with scroll state, always-visible Book Now and Directions

## 🚀 Deployment

### Option 1 — Netlify (Recommended)
1. Drag and drop `kings-crown.html` into [netlify.com/drop](https://app.netlify.com/drop)
2. Site is live instantly on a Netlify subdomain
3. Connect a custom domain in Netlify settings if desired

### Option 2 — GitHub Pages
1. Push `kings-crown.html` to a GitHub repository
2. Rename it to `index.html`
3. Go to **Settings → Pages → Deploy from branch**
4. Select `main` branch and save — live in ~60 seconds

### Option 3 — Any Static Host
Upload `kings-crown.html` (renamed to `index.html`) to any static hosting provider — Vercel, Cloudflare Pages, or shared hosting via FTP.
